# Blockchain-App
Blockchain App built in laravel app
## Project setup
1- Download the .zip folder 
2- Connect your database in .env file and run the following
```
php artisan migrate
```
```
php artisan db:seed --class=WalletSeeder
```

### Run
```
php artisan serve
```
